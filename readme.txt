Shutdown Tool V 0.1β
ただ、シャットダウンするのを簡単にしたいなーという思い付きで作ったソフト

==動作環境==
Windows 10
多分うごくであろう
WIndows 8/8.1
Windows 7
Windows Vista

==使い方==
Launcher.exeを起動して、実行したい動作を選択するだけです

==インストール/アンインストール==
インストールは、7zファイルを展開するだけです
アンインストールは、展開したフォルダを削除するだけです
レジストリは一切いじっていません

==注意事項==
7zファイルの展開は、展開したいフォルダの1個下のディレクトリにしてください
