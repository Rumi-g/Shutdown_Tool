Shutdown Tool V 0.2β
ただ、シャットダウンするのを簡単にしたいなーという思い付きで作ったソフト

==動作環境==
Windows 10
多分うごくであろう
WIndows 8/8.1
Windows 7
Windows Vista

==使い方==
Launcher.exeを起動して、実行したい動作を選択するだけです

==インストール/アンインストール==
setup.exeをダウンロードしたら、起動して画面の指示に従って、インストールしてください
アンインストールは、コントロールパネルから
